Simple Digital Dice Roller

A lightweight, interactive Python application that simulates rolling a standard six-sided die. This project demonstrates foundational programming concepts such as loops, user input handling, and the use of built-in libraries.

## Features
* **Interactive Interface:** Users can roll the dice repeatedly without restarting the program.
* **Randomized Results:** Uses Python's `random` module to ensure fair and unpredictable rolls.
* **Simple Exit:** Includes a clear way for the user to quit the application at any time.

## How to Run
1. Ensure you have **Python 3** installed on your computer.
2. Download the `dice_roller.py` file.
3. Open your terminal or command prompt.
4. Navigate to the folder where the file is saved.
5. Run the following command:
   ```bash
   python dice_roller.p
